### 👋 소개
임베디드 펌웨어와 엣지 AI를 연결해 문제를 끝까지 해결하는 엔지니어입니다.  
보드는 **STM32/ESP32**, OS는 **FreeRTOS/Zephyr**, 통신은 **UART·SPI·I2C·CAN·BLE·Wi-Fi**,  
AI는 **PyTorch·QAT·ONNX**를 중심으로 **현장에 올라갈 수 있는 결과물**을 만듭니다.

### ⚙️ Embedded Focus
- **RTOS & Drivers**: FreeRTOS, Zephyr, HAL/LL, 인터럽트/타이머, PM(저전력), 부트로더/OTA  
- **Connectivity**: BLE, Wi-Fi, MQTT, 소켓/REST, 펌웨어 OTA 파이프라인 구축  
- **Peripherals**: UART · SPI · I2C · CAN · I²S · ADC/DAC · 센서 드라이버 작성  
- **Debug/Tooling**: JTAG/SWD, OpenOCD, J-Link, Logic Analyzer, PlatformIO, CMake  
- **Edge AI**: PyTorch, QAT(정량화 학습), ONNX 변환/경량화, Jetson/RPi 실시간 추론
